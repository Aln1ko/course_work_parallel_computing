Проект веб-серверу з реалізацію інвертованого індексу
Для того щоб завантажити використати команду
 git clone https://github.com/Aln1ko/course_work_parallel_computing.git

В репозиторії знаходиться сам сервер на с++, кліента на с++ та кліент на js.
Для запуску серверу та кліенту на с++ необхідно відкрити проект та скомпілювати код, для запуску кліенту на js необхідно спочатку запустити файл proxy.js за допомогою команди node proxy.js, а після цього треба відкрити в браузері html файл(можна за допомогою live server в vscode, або іншими засобами)
